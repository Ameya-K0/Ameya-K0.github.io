Submarine Data Acquisition Module

I developed the firmware for a critical component of a data acquisition module used in a one-manned submarine. This system utilized a BNO055 IMU sensor to read real-time accelerometer and gyroscopic data, providing crucial insights into the submarine's movement.

The sensor was integrated with an nRF52840 semiconductor, enabling efficient data collection and processing. Using this data, I controlled a long strip of LEDs attached to the outside of the submarine, which visually indicated key motion parameters such as:

  • Tilting Detection – Displayed whether the submarine was leaning at an unsafe angle.
  • Speed Monitoring – Indicated whether the submarine was moving too fast or too slow.

This real-time feedback system provided the pilot with an intuitive, external status display, improving situational awareness and enhancing overall safety.

Project 2: RISC (Reduced Instruction Set Computer) Machine

❮ ❯

I designed and implemented a simple RISC machine capable of read-write memory operations and arithmetic processing. The system follows a structured pipeline where:

  • Instruction Fetch & Decode: The machine retrieves an instruction, which is then decoded by a finite state machine (FSM).
  • Datapath Execution: The instruction flows through the datapath, where the desired computation is performed using Boolean logic and a combinational ALU (Arithmetic Logic Unit).
  • Memory Operations: The computed result is stored in memory for future use.

I developed the project in Verilog using Visual Studio Code and compiled it on Quartus. I verified functionality by creating a testbench and running simulations in ModelSim to ensure that each instruction produced the expected output. Finally, I deployed the design onto an FPGA board (DE1-SOC) to demonstrate the working RISC machine in real hardware.

This project was one of my first experiences with hardware design and Verilog, making it an exciting and valuable learning experience! 🚀
